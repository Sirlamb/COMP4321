About this project (Phase 1)
This is a search engine implementation. 
It consists of 3 main parts: web crawler, retrivel database system, and search engine website.
But it only consist of 2 parts for now: crawler, database system 

Crawler: the spider get information from web and generate a database file
database system: the test code read a .db file to a .txt file

How to use
The project is based on Python, version 3.10 or above

To run the program, users should do the following.

install the required packages

pip install -r requirements.txt

To execute the program
python main.py
